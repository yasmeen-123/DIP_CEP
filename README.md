This project implements a Digital Image Processing (DIP)-based system for real-time detection of road lanes and objects from video input using Python.

The system processes each video frame using fundamental DIP techniques such as grayscale conversion, edge detection, region of interest masking, and contour analysis to identify lane boundaries. In parallel, object detection is performed to recognize and highlight dynamic entities present on the road.

The goal of this project is to demonstrate how classical image processing methods can be effectively applied to solve real-world problems like autonomous driving assistance and traffic monitoring, without relying entirely on deep learning models.
